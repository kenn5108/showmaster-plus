🔗 Connexion RocketShow API
Installation locale Raspberry Pi : ShowMaster+ tourne sur le Pi, accède à RocketShow via localhost:8080.
Serveur distant : ShowMaster+ tourne sur un autre serveur, renseigne l'IP ou le nom d'hôte du Pi.
📋 URLs d'accès
http://rocketshow.local:3000
http://rocketshow.local:3000/lyrics
Ouvrez l'URL du prompteur sur votre tablette. La page /lyrics s'adapte automatiquement à la taille de l'écran. Elle affiche les paroles en temps réel, synchronisées avec RocketShow.
⚙ QLC+ (prochainement)
QLC+ expose un WebSocket sur le port 9999. La synchronisation des effets lumière sur la timeline des chansons sera disponible dans une prochaine version.
